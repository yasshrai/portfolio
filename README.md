## üöÄ Getting Started

### Prerequisites

- bun

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yasshrai/portfolio.git
   cd portfolio
   ```

2. Install dependencies
   ```bash
   bun install
   ```

3. Run the development server
   ```bash
   bun run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
